Notes for the web site

This webapp is a project made for Coursera Django for Everybody, and added some other features after the course.

Tags: Python, Django, Bootstrap, css, js, fontawsome, avatar, search, filter

Logged in users can:

  • add an ad.
  • comment on ads.
  • send encrypted in-app messages to the ad owner.
  • use emojis on ad title, comments and messages.
  • use photo as avatar, default gavatar.
  • like ads and comments.
  • rate others ads.
  • edit/delete own ads and delete own comments.
  • send email to get reset password link.

Admin can manage users and ads on admin panel.

Github account can be used to login (social login).

Email activation is required to use the app.

Email confirmation is required to delete the account.

Photos (5 max) can be added to ads, no size limit but they are saved as compressed to max 1MB in the app.

Search function works through all ad titles and ad details.

Filter feature added.

The app is responsive and works on all devices.
